# Analisis dan Peramalan Inflasi Kota Mataram Menggunakan ARIMA

## Deskripsi
Proyek ini bertujuan untuk menganalisis dan meramalkan inflasi Kota Mataram menggunakan metode ARIMA (Autoregressive Integrated Moving Average). Analisis meliputi preprocessing data, uji stasioneritas, pemodelan ARIMA, evaluasi model, dan peramalan periode mendatang.

## Dataset
- Data inflasi bulanan Kota Mataram
- Periode: Januari 2007 – Oktober 2025

## Tools
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Statsmodels
- Scikit-Learn
- Plotly

## Metodologi
1. Import dan preprocessing data
2. Uji stasioneritas ADF
3. Pembagian data training dan testing
4. Pemodelan ARIMA
5. Rolling Forecast
6. Evaluasi MAE, RMSE, dan sMAPE
7. Forecast periode mendatang

## Hasil
Model ARIMA digunakan untuk menghasilkan prediksi inflasi Kota Mataram berdasarkan pola historis data.
